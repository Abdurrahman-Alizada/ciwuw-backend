# CIWUW Backend

**Call It What U Wanna (CIWUW)** is an e-commerce platform and application. This repository contains the Node.js/Express backend API that powers both the customer-facing web application and the administrative dashboard. 

The backend is built with TypeScript and uses MongoDB (via Mongoose) for data storage. It provides robust endpoints for user authentication, product management, order processing, and includes integrations with Stripe for secure payments and Cloudinary for media management.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Set up your `.env` file based on the required environment variables.
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
